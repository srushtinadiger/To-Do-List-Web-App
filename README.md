## ğŸ’» How to Use

1. **Adding a Task**
   - Type your task in the input field
   - Click "Add Task" or press Enter

2. **Completing a Task**
   - Click the checkbox next to a task to mark it as complete

3. **Deleting a Task**
   - Hover over a task to reveal the delete button
   - Click "Delete" to remove the task

4. **Filtering Tasks**
   - Click "All" to see all tasks
   - Click "Active" to see only incomplete tasks
   - Click "Completed" to see only completed tasks

5. **Clearing Completed Tasks**
   - Click "Clear Completed" to remove all completed tasks at once
